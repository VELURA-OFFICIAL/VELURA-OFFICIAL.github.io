Политика конфиденциальности для приложения "ТЬМА"

Дата вступления в силу: 16 ноября 2025 г.

1. Сбор и использование данных

Приложение "ТЬМА" ("Приложение") является инструментом для личного использования и не собирает, не хранит и не передает какие-либо персональные данные пользователя.

2. Взаимодействие со сторонними сервисами

Для выполнения своей функции Приложение взаимодействует с API TikTok. Взаимодействие с TikTok регулируется Политикой конфиденциальности TikTok. Мы не получаем и не храним ваши учетные данные от TikTok, используя стандартный протокол авторизации OAuth 2.0, предоставляемый платформой.

3. Безопасность

Приложение использует ключи доступа (API keys), выданные TikTok for Developers, для аутентификации запросов. Эти данные хранятся в безопасной среде и не передаются третьим лицам.
